🔐 Segurança Digital com Matemática

Este projeto foi desenvolvido para demonstrar como a matemática pode ser usada para melhorar a segurança digital, criando senhas fortes e analisando combinações com conceitos de análise combinatória.

📌 Objetivo

  • Gerar senhas seguras com Python
  • Calcular combinações possíveis com análise combinatória
  • Conscientizar sobre boas práticas de segurança digital

📄 Código Python (resumo)

import random
                                                                                                                                                                                                                                            import string

                                                                                                                                                                                                                                            def gerar_senha(tamanho):
                                                                                                                                                                                                                                                caracteres = string.ascii_letters + string.digits + string.punctuation
                                                                                                                                                                                                                                                    senha = ''.join(random.choice(caracteres) for _ in range(tamanho))
                                                                                                                                                                                                                                                        return senha

📚 Conceito Matemático

Fórmula usada para calcular o número de combinações possíveis de uma senha:

total_de_combinacoes = n_caracteres ** tamanho_da_senha

Quanto maior o número de combinações, mais difícil será quebrar a senha.
